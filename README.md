# E-commerce Data Pipeline: Engineering Phase

## 📊 Про проект
Цей проект імітує реальну роботу Analytics Engineer. Головна мета — побудувати автоматизований шлях даних (Pipeline) від "сирих" логів продажів до професійного бізнес-дашборду.

## 🛠 Технологічний стек
- **Python (Pandas):** Очищення даних, об'єднання таблиць, обробка помилок та конвертація валют.
- **SQL:** Моделювання архітектури бази даних за схемою "Зірка" (Star Schema).
- **Power BI:** Візуалізація ключових бізнес-метрик (Retention, AOV, Revenue Growth).

## 📂 Структура репозиторію
- `data/raw/`: Вхідні "брудні" дані.
- `data/processed/`: Очищені дані, готові до аналізу.
- `scripts/`: Python-скрипти для ETL-процесів.
- `sql/`: Запити для створення таблиць та аналітики.
